How Planr Works

01

Gathering

Planr auto-scrapes documents and websites from all councils, consolidating planning rules and historical application decisions into a single chatbot.

02

Processing

Information is embedded and indexed automatically with ChromaDB. We leverage the open-source Ollama LLM for advanced language understanding and response generation.

03

Returning

Planr uses Retrieval-Augmented Generation (RAG), letting users cross-reference chatbot responses against recent planning applications in any area.
